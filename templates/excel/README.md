# Excel模板使用说明

## 📋 模板下载

访问管理员页面：http://localhost:8080/admin

点击对应的模板下载按钮：
- **用户模板** - 下载用户数据导入模板
- **班级模板** - 下载班级数据导入模板  
- **课程模板** - 下载课程数据导入模板

## 📊 模板格式说明

### 1. 用户数据模板 (用户数据导入模板.xlsx)

| 列名 | 说明 | 示例 | 必填 |
|------|------|------|------|
| 学号/工号 | 唯一标识 | T001, S001, admin, 00005642 | ✅ |
| 姓名 | 用户姓名 | 张老师, 张三 | ✅ |
| 角色 | 用户角色 | student, teacher, admin | ✅ |

**示例数据：**
```
T001        张老师    teacher
00005642    李老师    teacher  
S001        张三      student
S002        李四      student
admin       管理员    admin
```

### 2. 班级数据模板 (班级数据导入模板.xlsx)

| 列名 | 说明 | 示例 | 必填 |
|------|------|------|------|
| 班级编号 | 6位数字标识 | 202101, 202102 | ✅ |
| 班级名称 | 班级完整名称 | 计算机2021-1班 | ✅ |
| 班级人数 | 班级学生总数 | 30, 28 | ✅ |

**示例数据：**
```
202101    计算机2021-1班    30
202102    计算机2021-2班    28
202201    软件2022-1班      32
```

### 3. 课程数据模板 (课程数据导入模板.xlsx)

| 列名 | 说明 | 示例 | 必填 |
|------|------|------|------|
| 课程名称 | 课程完整名称 | 工程实践B, 数据结构与算法 | ✅ |
| 授课老师工号 | 必须是已导入的老师 | 00005642, T001 | ✅ |
| 上课班级 | 使用班级名称 | 工程实践B18, 计算机2021-1班 | ✅ |
| 上课地点 | 教室位置 | 实验4-211, 教学楼A101 | ❌ |
| 课程日期 | 格式：yyyy-MM-dd | 2024-01-15 | ✅ |
| 上课时间段 | 格式：HH:mm-HH:mm | 08:00-11:40 | ✅ |
| 课程周次 | 对应学生课表周次 | 5, 8 | ❌ |

**示例数据：**
```
工程实践B        00005642    工程实践B18    实验4-211    2024-01-15    08:00-11:40    5
工程实践B        00005642    工程实践B18    实验3-404    2024-01-15    14:00-17:40    8
数据结构与算法    T001        计算机2021-1班    教学楼A101    2024-01-15    08:00-09:40    1
Java程序设计      T002        计算机2021-2班    教学楼B201    2024-01-15    10:00-11:40    1
```

## 🔄 导入流程

### 1. 准备数据
1. 下载对应的Excel模板
2. 按照模板格式填写数据
3. 确保数据格式正确

### 2. 上传导入
1. 访问管理员页面
2. 选择对应的导入区域
3. 点击"选择文件"或拖拽文件
4. 点击"开始导入"

### 3. 查看结果
系统会显示导入结果：
```
导入完成！成功：25条，失败：3条
```

## ⚠️ 注意事项

### 数据格式要求
- **日期格式**：必须为 `yyyy-MM-dd`，如 `2024-01-15`
- **时间格式**：必须为 `HH:mm-HH:mm`，如 `08:00-09:40`
- **班级编号**：必须为6位数字
- **角色**：只能是 `student`、`teacher`、`admin`

### 数据关联要求
- **课程导入**：授课老师工号必须是已导入的老师
- **课程导入**：上课班级必须是已导入的班级编号
- **唯一性**：学号/工号、班级编号不能重复

### ⚠️ 重要说明：工号格式
- **支持格式**：系统支持各种工号格式，包括带前导零的格式
- **示例**：`00005642`、`T001`、`S001`、`admin` 等
- **Excel处理**：如果工号是纯数字格式（如00005642），请在Excel中将其设置为文本格式，或在单元格前加单引号（'00005642）
- **自动处理**：系统已优化Excel导入功能，会自动保持工号的原始格式，包括前导零

### 导入顺序建议
1. 先导入用户数据（老师和学生）
2. 再导入班级数据
3. 最后导入课程数据

## 🚀 快速开始

### 测试数据导入
1. 访问：http://localhost:8080/admin
2. 下载"用户模板"
3. 填写测试数据：
   ```
   T001    张老师    teacher
   S001    张三      student
   ```
4. 上传导入
5. 重复步骤导入班级和课程数据

### 验证导入结果
1. 访问：http://localhost:8080/login
2. 使用测试账号登录：T001
3. 查看老师端功能是否正常

## 📞 技术支持

如遇到导入问题，请检查：
1. Excel文件格式是否为.xlsx
2. 数据格式是否符合要求
3. 必填字段是否完整
4. 关联数据是否存在

更多问题请联系开发团队。
